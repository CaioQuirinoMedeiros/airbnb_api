# Airbnb API

This project is a Airbnb's like app made with AdonisJS. It features authentication, CRUD and image upload, plus a beautiful map.

[Check the front-end](https://github.com/CaioQuirinoMedeiros/airbnb_web)

## Setup

Use `adonis serve --dev` to run in development environment.

Use `adonis migration:run` to run migrations
